何为代理？现实中的中介就是代理。例如：火车票代卖点。我们只需通过火车代卖点，就可以买到车票。

程序中的代理分两种：静态代理和动态代理。静态代理是设计时预先定义的预先。也就是编译期就存在的，由用户预先显式定义的。动态代理是运行期生成的。
